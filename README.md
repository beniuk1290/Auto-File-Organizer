# Auto File Organizer

Prosty skrypt w Pythonie do automatycznego organizowania folderu Downloads.

## Jak pobrać

1. Kliknij zielony przycisk **Code**
2. Wybierz **Download ZIP**
3. Rozpakuj plik ZIP na komputerze

## Jak uruchomić

1. Zainstaluj Python:
   https://www.python.org

2. Otwórz folder z plikami

3. Kliknij prawym przyciskiem myszy w folderze i wybierz:
   **Otwórz w terminalu** (lub PowerShell)

4. Wpisz:
   python auto_organizer.py

## Jak działa

- Monitoruje folder Downloads
- Automatycznie sortuje pliki do folderów (Images, Documents itd.)
- Dodaje datę do nazw plików
